# 康复助手 - 脑梗康复训练记录应用

这是一个专门为脑梗康复患者设计的康复训练记录应用，帮助患者和家人记录和管理每日的康复训练进度。

## 功能特点

### 📝 训练项目
- **慢走训练** - 记录每日步行时长
- **双手握拳平举** - 记录组数和次数
- **眼睛凝视训练** - 记录从左到右的次数

### 🎯 核心功能
1. **训练记录** - 每日记录康复训练数据
2. **目标设置** - 自定义每个项目的训练目标
3. **进度追踪** - 实时查看今日训练进度
4. **统计分析** - 查看训练天数、总记录数等统计信息
5. **训练提醒** - 设置每日定时提醒（使用浏览器通知）
6. **数据管理** - 导出数据、清空记录等管理功能

### 💡 特色
- 简洁的界面设计，适合老年用户使用
- 支持深色模式
- 响应式设计，支持手机、平板和电脑
- 数据本地存储，保护隐私
- 根据完成情况提供鼓励性反馈

## 技术栈

- **框架**: Next.js 16 (App Router)
- **UI 组件**: shadcn/ui + Tailwind CSS
- **图标**: Lucide React
- **数据存储**: localStorage
- **语言**: TypeScript

## 安装和运行

### 开发环境

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 在浏览器中打开 [http://localhost:3000](http://localhost:3000)

### 生产构建

1. 构建应用：
```bash
npm run build
```

2. 启动生产服务器：
```bash
npm start
```

## 使用说明

### 首次使用

1. 打开应用后，建议先进入"设置"页面
2. 设置适合你的训练目标
3. 启用并设置训练提醒时间
4. 授予浏览器通知权限（可选）

### 日常使用

1. **记录训练**
   - 点击首页的"开始记录"按钮
   - 填写完成的训练项目和数量
   - 点击"保存记录"

2. **查看进度**
   - 首页会显示今日的训练进度
   - 完成的项目会显示绿色标记和进度条

3. **查看统计**
   - 进入"统计"页面查看整体训练数据
   - 查看最近的训练记录

## 数据存储

所有训练数据都保存在浏览器的 localStorage 中，包括：
- 训练记录
- 用户目标设置
- 提醒设置

**注意**：清除浏览器数据会导致所有记录丢失。建议定期使用"导出数据"功能备份训练记录。

## 浏览器兼容性

推荐使用现代浏览器：
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+

## 隐私说明

- 所有数据仅存储在您的本地浏览器中
- 不会上传到任何服务器
- 不会收集任何个人信息
- 完全离线可用
